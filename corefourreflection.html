Core Four

Variable

Definition: Storage location paired with an associated symbolic name (an identifier), which contains some known or unknown quantity of information referred to as a value.

Examples: Python - name = ria count = 100 Scratch -


Function

Defintion: A procedure or routine (named section of a program) that performs a specific task. It is pre-defined so that it can be used multiple times in the same program.

Examples: Python - def sign(num): if sign(num)>0: return 1 elif num == 0: return 0 else: return -1 print sign(-3) Scratch -


Conditional

Defintion: A statement that can perform different computations or actions depending on whether a programmer-specified boolean condition evaluates to true or false.

Examples: Python - if number > 12 or number <= 0: print("error-shape too big") else: shape = 360/number Scratch -


Loops

Defintion: A sequence of instructions that is continually repeated a certain number of times or until a certain condition is, or is not, reached.
